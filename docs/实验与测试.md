# 实验与测试

## 1. 实验目的说明

实验与测试主要用于验证系统第一阶段设计中各模块的基本可用性，包括日志接收、解析、检测与告警展示流程。

---

## 2. 测试环境说明

- 操作系统：Windows / Linux
- 后端服务：FastAPI
- 数据库：MySQL
- 前端框架：Vue 3
- 通信方式：HTTP 与 WebSocket

---

## 3. 功能验证过程描述

### 3.1 日志接收测试

通过向系统持续发送模拟日志，验证日志接收接口的稳定性。

---

### 3.2 日志解析测试

发送符合 SSH 登录失败格式的日志，观察系统是否能够正确提取关键信息。

---

### 3.3 检测与告警测试

在短时间内发送多条相同来源 IP 的失败日志，验证检测规则是否被触发，并生成对应告警。

---

## 4. 后续测试规划

在后续功能扩展过程中，将进一步引入：

- 多规则并行测试；
- 高并发日志压力测试；
- 告警准确性与误报分析。
